Who we are

BIM Ops Studio is a small AEC-focused software studio led by individual developer Weber Gouin. We build Revit add-ins that automate construction documentation, improve coordination, and embed practical AI assistance directly into Revit workflows.

Active add-ins

ReviTask — AI-assisted documentation: batch sheet creation, smart view placement, auto-dimensioning/tagging, code-check helpers, and PDF/DWG export.

SmartSheetList — Sheet & detail management: preview drafting/detail views from libraries, batch load into active project, naming rules and standards enforcement.

RIACCA — Revit Intelligent Assistant for Code & Coordination: rulesets for IBC/FBC checks, issue/QC tracking, markup suggestions, and a PDF/QC pipeline.

Revit versions & API surface

  • Versions: Revit 2024–2026
  • APIs: Autodesk.Revit.DB, Autodesk.Revit.UI, IExternalCommand, ExternalEvent, DockablePane, FilteredElementCollector, view/sheet creation & export APIs

ADN usage policy

ADN-provided software is used strictly for development, testing, and marketing of the add-ins listed above. It is not used for production modeling or drafting.
