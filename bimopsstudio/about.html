About the Founder

I’m Weber Gouin — an architect turned developer who got tired of watching talented people waste hours on tasks a machine should handle.

After 15 years producing construction documents, coordinating BIM models, and managing Revit standards for architecture firms, I made a pivot: instead of using the tools, I started building them.

Today I build AI-powered automation for the AEC industry. My tools connect artificial intelligence directly to Autodesk Revit — so an architect can say "create the CD set" and have it actually happen.

What I Build

Revit API Development

Custom add-ins in C# and Python. WPF/DockablePane UIs inside Revit. pyRevit extensions and Dynamo scripts for production workflows. Named pipe communication bridges for real-time external application integration.

AI Integration for BIM

Classification systems that analyze BIM element properties and suggest categorizations. ML models with confidence scoring and human-in-the-loop review. Natural language interfaces that translate architect intent into Revit API operations.

Production Automation

Tools that turn hours of manual work into minutes. Sheet creation, view placement, annotation, code compliance checking, drawing set assembly. Tested with real architecture firms on real projects.
