Google Analytics 4 Setup for Download Tracking

Step 1: Create a Google Analytics Account

  1. Go to analytics.google.com
  2. Click "Start measuring"
  3. Create an account name (e.g., "BIM Ops Studio")
  4. Set up a property name (e.g., "BIM Ops Studio Website")
  5. Select your business details
  6. Choose "Web" as platform
  7. Enter your website URL: https://www.bimopsstudio.com
  8. Name your stream (e.g., "Main Website")

Step 2: Get Your Measurement ID

After setup, you'll get a Measurement ID that looks like: G-XXXXXXXXXX

Copy this ID - we'll use it in the code.

Step 3: Implementation Code

Add this to your products.html file (I'll do this for you once you have the ID):


<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
</script>

What You'll Be Able to Track:

  • Total downloads for each file
  • User location (country, city)
  • Download times and dates
  • User device and browser info
  • Which pages users visit before downloading
  • Return visitors vs new visitors

View Your Analytics:

Once set up, you can view all data at analytics.google.com

Downloads will appear under: Engagement > Events > file_download
