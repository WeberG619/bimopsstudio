## How to publish on GitHub Pages
1. Copy all files into your repo root.
2. Commit and push:
   ```bash
   git add .
   git commit -m "Split single-page site into multi-page site"
   git push
   ```
3. Ensure GitHub Pages is set to deploy from `main` branch, root folder.
4. Hard refresh your site.
